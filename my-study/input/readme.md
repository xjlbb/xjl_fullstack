# 表单
专门用来手机用户信息的

# 表单元素  ->还是HTML中的一些标签，只不过这些标签比较特殊
1. 什么是元素？
在HTML中 标签/标记/元素都是HTML中的标签
例如： <a> a标签/a标记/a元素

# 格式
<form>
    <表单元素>
</form>

# 常见的表单元素
input标签 有一个type属性，这个属性有很多类型的取值
    // 明文输入
    <input type="text">
    // 暗文输入
    <input type="password">
    // 单选框
    <input type="radio">
    //多选框
    <input type="checkbox">
